# Projeto 3 – Benchmark de Algoritmos de Ordenação

## Descrição

Este projeto tem como objetivo implementar e analisar o desempenho de algoritmos de ordenação, comparando resultados teóricos e experimentais.

Foram implementados os seguintes algoritmos:

- Insertion Sort
- Quick Sort

O foco do projeto é avaliar o comportamento dos algoritmos em diferentes cenários de entrada, considerando tempo de execução e análise estatística.

---

## Como executar

1. Clone o repositório:

```bash
git clone https://github.com/ghostlink359/TrabalhoGabriel.git
```
2. Acesse a pasta do Projeto:

```bash
cd projeto3-ordenacao
```

3. Execute o código:

```bash
python main.py
```
